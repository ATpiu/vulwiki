# Error Message

**漏洞等级：中危**

网站存在.net报错信息，由于网站未配置统一错误返回页面，导致aspx出错并显示出错误信息。

**可带来如下危害：**

1. 黑客可通过特殊的攻击向量，有可能泄漏如绝对路径、源代码、sql语句等敏感信息，恶意攻击者很有可能利用这些信息实施进一步的攻击。

**修复建议：**

1. 关闭PHP错误回显，或修正代码。
